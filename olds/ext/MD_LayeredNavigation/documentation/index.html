This tool lets you lure your visitors to the desired content of your website with the help of random linking images.

Features:

  • Extension will make simple SEO friendly URL
  • AJAX based filtering: No Page Reloads.
  • Price slider with auto update or submit button.
  • Multiple choice filter on product attributes.
  • Custom Routing suffix: You can choose you own.
  • Extremely Quick Navigation.
  • Multi-store and multi-language support.

Installation:

  • Prerequisite
    • Disable the cache:  Go to System > Cache  Management. Select all caches and select “Disable” from the actions dropdown and then hit submit button.
    • In case the compilation is on, Go to System > Tools > Compilation and disable it.
  • Follow the steps to install the extension.
     
  1. Upload or copy all folders in the zip package to your Magento site root folder. Select to merge the folder in case it is asked.
  2. After the successful upload, log in your Magento admin panel to refresh cache. Go to System/Cache Management. Select all caches and take action “Refresh” then submit.  In case you are already logged in,  kindly logout of the admin panel relogin.
  3. Navigate to System/Configuration, to see “Layered Navigation Settings” under the “MD” tab installed successfully.
  4. Thereafter proceed with the configuration of the extension and set the values as per your choice.
  5. Check the frontend of your webstore to see the extension in action. 

How to Use:

System Configuration Setting

  • Log in admin panel and configure System > MAGEDRIVE > Layered Navigation Settings
  • In Enable choose Yes
  • Enable ajax: Choose yes if you doesn't want to reload the page.
  • Enable Price slider: Set yes for display in layered navigation.
  • Price slider submit type: Delay and sumit option will autho submit after x seconds.
  • Price slider delay: Set seconds for delay and auto submit option.
  • Enable Multipe Choise Filter: Choose yes for Multiple attribute filter.
  • Routing Souffix: Choose you own.
